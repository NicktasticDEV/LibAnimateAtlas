A library made for loading Adobe Animate Texture Atlas files for any purpose, whether it's for manipulating or rendering Texture Atlas data!

> [!WARNING]
> This library is currently not ready for production use and is still under development.

## Plans

- Built-in support for OpenGL
- Support for the following versions of Texture Atlas
  - [Adobe Animate](https://wiki.starling-framework.org/extensions/adobe-animate)
  - [BetterTextureAtlas](https://github.com/Dot-Stuff/BetterTextureAtlas)
